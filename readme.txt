== Frequently Asked Questions ==

= How can I insert poll into my pages/posts =

Just add Poll Builder shortcode (Ex [ypl_poll id=2725] where id is your poll id ) to your pages/posts.


= What files I need to upload for installing Poll builder plugin =

You need to select Poll builder .zip file, there is no need to extract the zip file, just upload it.
